# ATM System Project

This is a simple ATM (Automated Teller Machine) system developed using Visual Studio and C#.

## About the Project
This project simulates the basic functions of an ATM machine, allowing users to interact with their account in a simple and structured way.

## Features
- Check account balance
- Deposit money
- Withdraw money
- Simple menu-based system
- Beginner-friendly C# code

## Purpose
The goal of this project is to practice programming concepts such as variables, conditions, loops, and user input, as well as to learn how to use Git and GitHub.

## How to Run
1. Open the project in Visual Studio
2. Build the solution
3. Run the program (Start button)
4. Follow the on-screen menu
   
## Notes
This project is for learning purposes only and does not connect to real banking systems.
